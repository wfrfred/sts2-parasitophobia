# Parasitophobia

## Description

A Slay the Spire 2 mod that replaces parasites with seals:

- The card "Infection" now uses the Bai Xi expression image as its card face.
- The monster "Phrog Parasite" model and animations are replaced with seal.
- The monster "wriggler" model and animations are replaced with little seals.

## Install(Windows)

Download the latest release archive.
Extract it, and place the entire folder inside the game's mods directory:

``` text
root 
├── mods
│   ├── ParasitePhobia
│   │   ├── ParasitePhobia.dll
│   │   ├── ParasitePhobia.json
│   │   └── ParasitePhobia.pkg
```

If the mods folder does not exist, please create it manually.

For first-time installation, note that modded save files are separate from the original game saves.
- When you launch the game after installing the mod, allow loading mods.
- Copy your profilex folder from %AppData%\SlayTheSpire2\steam\<SteamID> into the modded folder to transfer your original save files to the modded save.
